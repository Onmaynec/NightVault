# Current Release: NightVault 1.3.9

**NightVault 1.3.9 — Messenger Features Update**

Цель версии: усилить клиентские функции мессенджера и закрыть визуальные/адаптивные проблемы админки, найденные после 1.3.8.

Главное:
- forward / multi-select / saved messages;
- media history / mentions / notifications / DND;
- paste & drag/drop файлов;
- profile 2.0 / block / reports;
- исправления Settings, Updates, Tests, Sessions, Logs/Dashboard overflow в админке;
- совместимый импорт backup из прошлых версий.
