# NightVault 1.3.7 — Performance & UX Update

Версия 1.3.7 концентрируется на стабильности интерфейса и ощущении качества: частичный рендер, WebSocket-first live events, кастомные голосовые, стабильный photo viewer, Crimson 2.0, backup package v2 и усиленная админ-консоль.

## Главное

- Partial render fingerprints для chats/messages/contacts/profile/right panel.
- WebSocket-first live update pipeline, polling оставлен fallback.
- Полностью кастомный voice player без native audio controls.
- Единый Media Viewer Service с zoom/drag/next/prev.
- Сохранение focus/cursor/scroll/voice state между обновлениями.
- Profile color влияет на аватар, подсветку и панели.
- Profile banner hydrator/fallback и кнопка удаления баннера.
- Group visual card/avatar/settings улучшены.
- Contacts targeted updates и уведомления о заявках.
- Admin log filters, расширенные команды и backup package v2.
- Release audit для GitHub Actions/autoupdater.
- Crimson 2.0 и UI-kit размеров кнопок/форм.
