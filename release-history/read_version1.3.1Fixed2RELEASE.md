# NightVault 1.3.3

- Исправлен `nv120CollectLinks is not defined`, из-за которого ломалась отрисовка центра и загрузка баннера.
- Добавлен общий E2EE key vault между локальными профилями клиента: повторный вход и несколько клиентов на одном ПК не создают новый несовместимый ключ для того же аккаунта.
- Исправлено действие выбора файла во вкладке “Файлы” (`localStorage.nv131SelectedFile=...`).
- Исправлено исчезновение сообщений после реакции: теперь реакция применяется оптимистично и сохраняет расшифрованный payload.
- Контекстное меню ПКМ больше не уходит за пределы экрана.
- Голосовые сообщения получили нормальный вид в стиле NightVault.
- Поправлены кнопка сохранения и блок “Обзор” в настройках.
