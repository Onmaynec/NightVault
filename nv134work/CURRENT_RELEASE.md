# NightVault 1.3.4

## Changes

- Default first-run theme: Crimson, red accent, crimson chat background.
- Server Admin now launches the server on `0.0.0.0:3000` for RadminVPN/LAN testers.
- Voice messages redesigned to match compact Telegram-like waveform bubbles in NightVault style, including voice-like `.webm/.ogg/.m4a/.mp3/.wav` audio attachments.
- Voice playback state survives chat updates/rerenders.
- Image attachments open in a full viewer with zoom, pan, reset and close controls.
- GitHub updater target remains `Onmaynec/NightVault`; use tag `v1.3.4`.

## Release assets for auto-update

Attach to GitHub release `v1.3.4`:

- `NightVault-Setup-1.3.4.exe`
- `NightVault-Setup-1.3.4.exe.blockmap`
- `latest.yml`
